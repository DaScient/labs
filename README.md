# Labs | DaScient, LLC | AI | Quantum | GenML

Welcome to **Labs** — the experimental playground of [DaScient](https://github.com/DaScient).  
This repository collects prototypes, tools, and research projects spanning data science, AI, web automation, and systems engineering.

---

## 📂 Directory Overview

- **assets/** — shared images, datasets, and supporting files used across experiments.  
- **cloudflare/** — Cloudflare Workers and edge function experiments.  
- **configs/** — configuration files and templates for services, scripts, and pipelines.  
- **crypto-dashboard/** — tools, dashboards, and visualizations for cryptocurrency tracking and analytics.  
- **functions/** — reusable serverless functions and micro-utilities.  
- **gozaddy/** — domain management and automation experiments with GoDaddy.  
- **instagram/** — scripts and bots for Instagram data, automation, or integrations.  
- **intel/** — intelligence gathering, scraping, and data pipelines.  
- **rag/** — Retrieval-Augmented Generation (RAG) prototypes and experiments.  
- **robovet/** — robotics, vetting systems, or automated evaluation frameworks.  
- **rss/** — RSS feed scrapers, processors, and syndication utilities.  
- **scripts/** — general-purpose scripts for automation, testing, and utilities.  
- **workers/** — distributed tasks, background jobs, and worker service prototypes.

## 🔧 Getting Started

### Clone the repo
```bash
git clone https://github.com/DaScient/labs.git
cd labs

Install dependencies

Each subproject may define its own environment or requirements.txt.
For example:

cd crypto-dashboard
pip install -r requirements.txt

Run an experiment

Navigate into a project folder and follow its specific README or run instructions:

cd rag
python main.py
